> Aside is mainly used to make the page hierarchical. Putting complex or deeper operations in the aside can make the main page more concise.

## Basic Usage

Aside controls whether it is turned on or off by the `isOpen` property, and the `onClose` property is used to clear the state, so both properties are required.

```jsx tabs
import { Aside, Button } from '@webapps-ui/core-react'

class AsideBasic extends React.Component {
  constructor(props) {
    super(props)
    this.state = { isOpen: false }
  }
  openAside() {
    this.setState({ isOpen: true })
  }
  onClose() {
    this.setState({ isOpen: false })
  }
  render () {
    return (
      <div>
        <Button ghost type="primary" onClick={() => this.openAside()}>OPEN ASIDE</Button>
        <Aside isOpen={this.state.isOpen} onClose={() => this.onClose()} title="Aside Title">
```

`Aside` actually generates two DOM nodes, one is Aside and the other is Backdrop which is passed to the body as its children through the `React Portal` API, but it should be noted that the final rendered context is the same as the Aside. Closing the Aside including clicking the Backdrop will automatically destroy both nodes.

## Aside with Spinner

You can use a Spinner nested inside an Aside which is also a common way to wait the asynchronous requests.

```jsx tabs

  constructor(props) {
    super(props)
    this.state = { isOpen: false, isLoading: true }
  }
  openAside() {
    this.setState({ isOpen: true })
    window.setTimeout(() => this.setState({ isLoading: false }), 500)
  }
  onClose() {
    this.setState({ isOpen: false, isLoading: true })
```

## Nested Asides

You should use nested Asides when the content of Aside is more complex or they have logical relationship, such as the content of the first Aside is a list item and it's another Aside when each item opens.

```jsx tabs
import { Aside, Button } from '@webapps-ui/core-react'

class NestedAsides extends React.Component {
  constructor(props) {
    super(props)
    this.state = { isOpen1: false, isOpen2: false, isOpen3: false }
  }
  openAside(number) {
    this.setState({ [`isOpen${number}`]: true })
  }
  closeAside(number) {
    this.setState({ [`isOpen${number}`]: false })
  }
  render () {
    return (
      <div>
        <Button ghost type="primary" onClick={()=>this.openAside(1)}>
          OPEN NESTED ASIDES
        </Button>
        <Aside isOpen={this.state.isOpen1} onClose={()=>this.closeAside(1)} title="Aside One">
          <Button ghost type="primary" onClick={()=>this.openAside(2)} block>
            OPEN SECOND ASIDE
          </Button>
          <Aside isOpen={this.state.isOpen2} onClose={()=>this.closeAside(2)} title="Aside Two">
            <Button ghost type="primary" onClick={()=>this.openAside(3)} block>
              OPEN THIRD ASIDE
            </Button>
            <Aside isOpen={this.state.isOpen3} onClose={()=>this.closeAside(3)} title="Aside Three">
              <Button type="primary" onClick={()=>this.closeAside(1)} block>
                CLOSE ALL ASIDES
              </Button>
            </Aside>
          </Aside>
        </Aside>
      </div>
    )
  }
}
```
