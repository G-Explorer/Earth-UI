## Type

The `default` and `primary` buttons are used in most cases, you can also use other types in specific scenarios.

```jsx tabs
import { Button } from '@webapp-suite/ui'

const ButtonType = () => {
  return (
    <>
      <Button>DEFAULT</Button>
      <Button type="primary">PRIMARY</Button>
      <Button type="accept">ACCEPT</Button>
      <Button type="warning">WARNING</Button>
      <Button type="danger">DANGER</Button>
      <Button type="text">TEXT</Button>
      <Button type="link">LINK</Button>
    </>
  )
}
```

## Ghost

The `ghost` style button is commonly used in the place where you don't want the button to stand out.

```jsx tabs
import { Button } from '@webapp-suite/ui'

const ButtonGhost = () => {
  return (
    <>
      <Button ghost>DEFAULT</Button>
      <Button ghost type="primary">PRIMARY</Button>
      <Button ghost type="accept">ACCEPT</Button>
      <Button ghost type="warning">WARNING</Button>
      <Button ghost type="danger">DANGER</Button>
      <Button ghost type="text">TEXT</Button>
      <Button ghost type="link">LINK</Button>
    </>
  )
}
```

## Loading Button

Assign the `loading` property to Button to initialize a spinner. Click event will be disabled when button is loading state, open the browser console you will see more logs.

```jsx tabs
import { Button } from '@webapp-suite/ui'
import { useState } from 'react'

const LoadingButton = () => {
  const [loading, setLoading] = useState(false)
  const handleClick = () => {
    console.log('clicked!')
    setLoading(true)
    setTimeout(() => setLoading(false), 2000)
  }
  return <Button type="primary" loading={loading} onClick={handleClick}>SHOW LOADING</Button>
}
```

## Block Button

The `block` property will make the button fit to its parent width.

```jsx tabs
import { Button } from '@webapp-suite/ui'

const ButtonBlock = () => {
  return (
    <>
      <Button type="primary" block>PRIMARY</Button>
      <Button block>DEFAULT</Button>
    </>
  )
}
```

## Size

Besides default `md` size, Button component provides three additional sizes for you to choose among different scenarios.

```jsx tabs
import { Button } from '@webapp-suite/ui'

const ButtonMicro = () => {
  return (
    <>
      <Button type="primary" size="sm">PRIMARY</Button>
      <Button size="sm">DEFAULT</Button>
      <br/><br/>
      <Button type="primary" size="md">PRIMARY</Button>
      <Button size="md">DEFAULT</Button>
      <br/><br/>
      <Button type="primary" size="lg">PRIMARY</Button>
      <Button size="lg">DEFAULT</Button>
    </>
  )
}
```

## Icon

Use icons to add more meaning to Button. You can use icon alone to save some space, or use it with text.

```jsx tabs
import { Button } from '@webapp-suite/ui'

const ButtonIcon = () => {
  return (
    <>
      <Button icon="statuschange" type="primary" />
      <Button icon="statuschange" type="primary" ghost />
      <Button icon="statuschange" />
      <Button icon="add" />
      <Button icon="statuschange" size="sm" />
      <Button icon="statuschange" ghost />
      <Button icon="activity">DATA ANALYSIS</Button>
      <Button icon="statuschange" type="text">Refresh</Button>
    </>
  )
}
```

## Disabled

The `disabled` attribute determines if the button is disabled.

```jsx tabs
import { Button } from '@webapp-suite/ui'

const ButtonDisabled = () => {
  return (
    <>
      <Button disabled type="primary">PRIMARY</Button>
      <Button disabled>DEFAULT</Button>
      <Button disabled type="accept">ACCEPT</Button>
      <Button disabled type="text">TEXT</Button>
      <Button disabled type="link">LINK</Button>
      <br/><br/>
      <Button icon="statuschange" disabled />
      <Button icon="activity" type="primary" ghost disabled>DATA ANALYSIS</Button>
    </>
  )
}
```
