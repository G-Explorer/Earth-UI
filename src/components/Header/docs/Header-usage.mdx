## Title

```jsx vertical
import { Header } from '@webapps-ui/core-react'

const HeaderTitle = () => {
  return (
    <Header title="Header" />
  )
}
```

## Icon

In the most scenarios, app logo/icon should be square, like following example shows:

```jsx vertical
import { Header } from '@webapps-ui/core-react'

const HeaderIconSquare = () => {
  return (
    <Header icon="https://cosmos-x.oss-cn-hangzhou.aliyuncs.com/blue-bg-webapps-ui-icon.png" title="Header" />
  )
}
```

In addition, you can also use rectangle logo:

```jsx vertical
import { Header } from '@webapps-ui/core-react'

const HeaderIconRectangle = () => {
  return (
    <Header icon="https://cosmos-x.oss-cn-hangzhou.aliyuncs.com/eb-logo.png" title="Header" />
  )
}
```

## Background color

```jsx vertical
import { Header } from '@webapps-ui/core-react'

const ColorfulHeader = () => {
  return (
    <Header icon="https://cosmos-x.oss-cn-hangzhou.aliyuncs.com/white-bg-webapps-ui-icon.png" title="Header" color="blue" />
  )
}
```

## Buttons

```jsx vertical
import { Header, Button } from '@webapps-ui/core-react'

const HeaderButtons = () => {
  return (
    <Header icon="https://cosmos-x.oss-cn-hangzhou.aliyuncs.com/blue-bg-webapps-ui-icon.png" title="Header">
      <Button>BUTTON</Button>
      <Button icon="logout" />
    </Header>
  )
}
```
