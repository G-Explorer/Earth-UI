## 基本功能

```jsx tabs
import { Button, Modal, ModalHeader, ModalBody } from '@webapp-suite/ui'

class ModalBasic extends React.Component {
  handleOpen() {
    this.refs.modal.open()
  }
  render() {
    return (
      <div>
        <Button onClick={() => this.handleOpen()}>Open</Button>
        <Modal ref="modal">
          <ModalHeader title="标题" />
          <ModalBody>
            内容...
          </ModalBody>
        </Modal>
      </div>
    )
  }
}
```

## 锁定的

点击半透明背景不会自动关闭

```jsx tabs
import { Button, Modal, ModalHeader, ModalBody } from '@webapp-suite/ui'

class ModalLock extends React.Component {
  handleOpen() {
    this.refs.modal.open()
  }
  render() {
    return (
      <div>
        <Button onClick={() => this.handleOpen()}>Open</Button>
        <Modal ref="modal" lock>
          <ModalHeader title="锁定的" />
          <ModalBody>
            点击半透明背景不会自动关闭
          </ModalBody>
        </Modal>
      </div>
    )
  }
}
```

## 嵌套的

```jsx tabs
import { Button, Modal, ModalHeader, ModalBody } from '@webapp-suite/ui'

class ModalNested extends Component {
  handleOpen() {
    this.refs.modal.open()
  }
  render() {
    return (
      <div>
        <Button onClick={() => this.handleOpen()}>Open</Button>
        <Modal ref="modal">
          <ModalHeader title="第一层模态框" />
          <ModalBody>
            <Button onClick={() => this.refs.modal2.open()}>
              打开第二层模态框
            </Button>
            <Modal ref="modal2" size="lg">
              <ModalHeader title="第二层模态框" />
              <ModalBody>
                第二层模态框内容...
              </ModalBody>
            </Modal>
          </ModalBody>
        </Modal>
      </div>
    )
  }
}
```
