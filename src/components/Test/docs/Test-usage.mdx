## Basic Usage

The display and hiding of loading effects are controlled by the value of `isLoading`.

```jsx tabs
import { Spinner, Switch } from 'earth-ui'

class SpinnerBasic extends React.Component {
  constructor() {
    super();
    this.state = { isLoading: false }
  }
  toggle(value) {
    this.setState({ isLoading: value });
  }
  render() {
    const border = {
      height: '120px',
      lineHeight: '118px',
      color: '#aaa',
      textAlign: 'center',
      border: '1px dashed #ccc'
    }
    return (
      <div>
        <div style={{ marginBottom: 16 }}>
          Open Spinner：<Switch on={this.state.isLoading} onChange={value => this.toggle(value)} />
        </div>
        <div style={border}>
          <Spinner isLoading={this.state.isLoading}>
            Will be overridden by the load effect
          </Spinner>
        </div>
      </div>
    )
  }
}
```

This component will ensure that the loading display time is at least `300ms` to avoid the bad experience as "flash".

## Custom description

The description of spinner could be changed by the `tip` property。

```jsx tabs
import { Spinner, Switch } from 'earth-ui'

class SpinnerTip extends React.Component {
  constructor() {
    super();
    this.state = { isLoading: false }
  }
  toggle(value) {
    this.setState({ isLoading: value });
  }
  render() {
    const border = {
      height: '120px',
      lineHeight: '118px',
      color: '#aaa',
      textAlign: 'center',
      border: '1px dashed #ccc'
    }
    return (
      <div>
        <div style={{ marginBottom: 16 }}>
          Open Spinner：<Switch on={this.state.isLoading} onChange={value => this.toggle(value)} />
        </div>
        <div style={border}>
          <Spinner tip="Loading..." isLoading={this.state.isLoading}>
            Custom description
          </Spinner>
        </div>
      </div>
    )
  }
}
```

## Dialog Type

The `danger` type dialog can be nested in the `warning` type dialog, so that the interaction will have a sense of hierarchy from weak to strong.

```jsx run
import { dialog, Button } from 'earth-ui'

function DialogIcon() {
  dialog.confirm('Proceed to Checkout?', 'CHECK OUT', {
    icon: 'cart'
  })
}
```
