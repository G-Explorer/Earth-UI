## Basic Usage

The target element is triggered if it satisfies the text-overflow condition (non-inline element and content overflow)

```jsx tabs
import { TextOverflow } from '@webapps-ui/core-react'

const TextOverflowBasic = () => {
  return (
    <TextOverflow>
      <p style={{width: '100px'}}>I am a very long text, the mouse over can show all.</p>
    </TextOverflow>
  )
}
```
